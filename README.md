# Number Guessing website
ZE Guess Number is a fun, interactive JavaScript game where users try to guess a secret number, showcasing DOM manipulation, dynamic feedback, and clean UI logic with responsive design.

## what i got from building this?
By exploring this project, you’ll see how I implement game logic, handle user interactions, and manage real-time feedback using core JavaScript concepts. It demonstrates my understanding of event handling, conditionals, and DOM updates while keeping the interface engaging and visually consistent. This project highlights my ability to combine functionality with a smooth user experience—an essential foundation for any interactive web application.

## Tech stacks 
 - HTML
 - CSS
 - JavaScript
